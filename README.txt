Rapports-Devoirs-poly

Ce repo contient un modèle LaTeX pour faire aisément des rapports de travaux
pratiques ou des devoirs à Poly.

Il suffit normalement de changer les définitions dans le préambule du
fichier "rapport.tex" et d'écrire son texte dans le corps du document
suite à la table des matières (qu'on peut enlever d'ailleurs).

Les auteurs voudront aussi mettre leurs références au format Bibtex
dans le fichier "rapport.bib".
